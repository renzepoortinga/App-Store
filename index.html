App Store

Nieuwe app toevoegen

Zo werkt deze persoonlijke app store — geïnspireerd op Naval's “A Return to Code”. Elke app is een zelfstandige web-app in dit repository; er is geen server en je data blijft op je eigen apparaat.
  1. Vraag een AI-agent (Claude Code) een app te bouwen in een nieuwe map apps/<naam>/ — één zelfstandig HTML-bestand, zonder externe afhankelijkheden.
  2. Voeg de app toe aan de lijst in index.html (het APPS-array onderaan).
  3. Publiceer via GitHub Pages en open deze pagina op je iPhone.
  4. Installeer: open een app → deel-knop → “Zet op beginscherm”. De app werkt daarna als een gewone app, ook offline.
